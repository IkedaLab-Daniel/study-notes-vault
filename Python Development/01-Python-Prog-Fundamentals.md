## Python Programming Fundamentals

### Course Purpose

* Introduces core Python concepts and programming fundamentals.
* Builds a foundation for more advanced topics in the certificate program.
* Includes practical exercises, projects, quizzes, and hands-on labs.

### Module 1: Introduction to Python

Topics:

* History and origins of Python
* Core features and applications
* Setting up a Python development environment
* Writing and running Python programs
* Basic debugging techniques

Hands-on Activity:

* Writing Python code with attention to detail
* Practicing basic debugging skills

### Module 2: Python Basics

Topics:

* Variables
* Data types
* Operators
* Control flow structures

Hands-on Activities:

* Creating and manipulating lists
* Using loops and variables
* Performing calculations and storing results

### Module 3: Functions and Modules

Topics:

* Functions
* Classes and objects
* Modules
* Code organization and reusability
* Modular programming

Hands-on Activity:

* Using Python built-in functions to solve tasks creatively

### Module 4: Data Structures in Python

Topics:

* Lists
* Tuples
* Dictionaries
* Sets
* Data structure selection and usage
* Sorting and searching algorithms

Projects:

1. Exercises using lists, tuples, dictionaries, and sets.
2. Applying sorting and searching algorithms for data management and retrieval.

### Module 5: Error Handling and Debugging

Topics:

* Identifying and handling errors
* Common Python exceptions
* Debugging tools and techniques
* Exception handling best practices

Hands-on Activity:

* Solving coding problems using debugging and exception handling techniques

### Module 6: Testing Basics & Version Control

Topics:

* Types of software testing
* Testing principles and processes
* Using pytest
* Writing and running test functions
* Git fundamentals
* Version control and collaboration

Final Project:

* Develop a script that performs data manipulation.
* Implement error handling.
* Demonstrate debugging techniques.
